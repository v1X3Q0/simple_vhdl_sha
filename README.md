## building stuff for quartus 13.0.1

Newer computers(ubuntu 24) requirements for getting the workspace up.

First build libpng12, download from [libpng source](https://libpng.sourceforge.io/index.html) or something.

```bash
sudo apt-get install libxext-dev:i386 \
 libfontconfig-dev:i386 libxrender-dev:i386 \
 libsm6:i386 libfreetype-dev:i386
```
